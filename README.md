# Sistema-de-Controle-de-Submiss-o-de-Atividades-Complementares

# Sistema desenvolvido com o intuito de auxiliar as insituições de ensino que demandem o envio, averiguação e aceitação de horas complementares para a conclusão de curso.
# A plataforma permite com que os alunos enviem certificados e documentos que sejam aceitos dentro das categorias disponíveis, estando de acordo com o limite horário máximo permitido por submissão.
# O sistema também oferece uma caixa de controle no perfil do usuário, permitindo com que suas horas complementares sejam visualizadas de maneira mais prática e direta.
